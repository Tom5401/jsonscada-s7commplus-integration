**Add public `WaitForAlarmNotification` method** at the end of the class (before the closing braces):
This method wraps the private `WaitForNewS7plusReceived` and `m_ReceivedPDU` so that external code (S7CommPlusClient's alarm thread) can receive alarm PDUs without accessing private members. It returns the raw Notification object, or null on timeout/error. The credit-limit check and replenishment logic is also included here since it requires access to `AlarmSubscriptionSetCreditLimit` and `m_AlarmNextCreditLimit`.

```csharp
/// <summary>
/// Waits for a single alarm notification PDU and returns the deserialized Notification.
/// Returns null on timeout or error (check m_LastError).
/// Also handles credit-limit replenishment internally.
/// </summary>
public Notification WaitForAlarmNotification(int waitTimeout)
{
    m_LastError = 0;
    WaitForNewS7plusReceived(waitTimeout);
    if (m_LastError != 0)
    {
        return null;
    }

    var noti = Notification.DeserializeFromPdu(m_ReceivedPDU);
    if (noti == null)
    {
        return null;
    }

    // Credit-limit replenishment (BUG-01 fixed: uses AlarmSubscriptionSetCreditLimit)
    if (noti.NotificationCreditTick >= m_AlarmNextCreditLimit - 1)
    {
        short creditLimitStep = 5;
        m_AlarmNextCreditLimit = (short)((m_AlarmNextCreditLimit + creditLimitStep) % 255);
        AlarmSubscriptionSetCreditLimit(m_AlarmNextCreditLimit);
    }

    return noti;
}
```
